**Why Synap?**
- Discover 1200+ AI tools across every category

- Real-time usage tracking and free-tier limit monitoring

- Cross-platform sync (mobile → web, favorites synced everywhere)

## 🎨 DESIGN SYSTEM: "DARK VOID" + AURORA EFFECTS

### **Brand Identity**
- **Name**: Dark Void (Glassomorphism with Aurora effects)
- **Aesthetic**: Interstellar, premium, modern, futuristic
- **Philosophy**: Every component feels "elevated" with gradients, glows, and glass effects

### **Color Palette**

| Element | Hex Code | RGB | Usage |
|---------|----------|-----|-------|
| Primary Cyan | #00C8E8 | 0, 200, 232 | Brand color, highlights, CTAs, accents |
| Cyan Light | #00D9FF | 0, 217, 255 | Lighter accents, hovers, glows |
| Magenta (Accent) | #FF00CB | 255, 0, 203 | Secondary accent, gradients |
| Dark Background | #05070C | 5, 7, 12 | Primary background |
| Darker Card BG | #030508 | 3, 5, 8 | Card/Container backgrounds |
| Text Primary | #F0F4F8 | 240, 244, 248 | Main text color |
| Text Secondary | #A0AEC0 | 160, 174, 192 | Subtext, muted info |
| Gradient 1 | Cyan → Magenta | - | Feature highlights |
| Gradient 2 | Dark Blue → Purple | - | Background accents |

**Gradient Usage Examples:**
- Hero section background: subtle cyan-to-magenta fade
- Featured tool cards: border gradient with inner glow
- CTA button: full cyan-to-magenta with hover animation

### **Typography**

| Element | Font | Weight | Size | Usage |
|---------|------|--------|------|-------|
| Headings (H1-H3) | Syne | 700-800 | 36px-48px (H1), 28px-32px (H2), 22px-24px (H3) | Main titles, headlines |
| Body Text | DM Sans | 400-500 | 14px-16px | Paragraphs, descriptions |
| Subtext | DM Sans | 400 | 12px-13px | Meta info, secondary copy |
| Code/Tech | JetBrains Mono | 400-500 | 12px | Code snippets, technical terms |

**Font Fallbacks**: Georgia serif if web fonts fail  
**Font Import**: Google Fonts API

### **Visual Effects**

1. **Glassomorphism**
   - Semi-transparent white overlay (5-10% opacity)
   - Subtle backdrop blur (8-12px)
   - Border: 1px solid rgba(255,255,255, 0.15)

2. **Aurora / Glow Effects**
   - Box-shadow with cyan/magenta glow on hover
   - Animated gradient background shifts
   - Floating particles or subtle animations

3. **Animations**
   - Fade-in on page load (250ms ease-out)
   - Hover scale (1.02) on cards
   - Border glow on interactive elements
   - Smooth state transitions (200-300ms)
